# Personal Finance & Expense Analytics System

A Core Java console application for managing expenses, income, budgets, and savings.

## Features

* Add Expense
* View Expenses
* Delete Expense
* Search Expenses by Category
* Search Expenses by Amount Range
* Total Expense Calculation
* Highest Expense Tracking
* Category-wise Analytics
* Stream-based Analytics
* Budget Management
* Income Tracking
* Savings Report
* Exception Handling
* File Handling

## Technologies Used

* Java
* OOP (Object-Oriented Programming)
* Collections Framework (ArrayList, HashMap)
* Java 8 Streams
* Exception Handling
* File Handling
* Git & GitHub
